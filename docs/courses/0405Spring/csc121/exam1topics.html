Exam Format

  • Writing methods and classes section
  • Explaining what source code accomplishes or giving the output of a section of source code
  • Short answer section, where short answer could involve the evaluation of an expression
  • Medium answer section:
    • may cover any concepts listed in preceding bullets, so answers may be short paragraphs, involving several sentences (e.g., book exercise 1.32)
    • may cover conceptual notions of: graphics, classes, objects, etc., with no writing of code
