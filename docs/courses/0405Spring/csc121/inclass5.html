Starting Project 4

Goals:
Working with project 4: Arrays of Points.
Getting started:
  • Copy the folder graffiti from the public folder on the I: drive to your class folder on the I: drive.
  • Start up BlueJ
The Steps of the Lab:
  1. Open the graffiti project (project 4).
    Read

    In project 4 you will be writing several methods in the Recognizer class. Open that class and study its fields and the constructor. You don't need to worry about the details of initializing the baseset from a file, but you should recognize that a nested loop is used to assign values to the points of each baseset 2-D array of Points.
    Do

    Complete steps 1 and 3 of project 4.
